# Week Vibes

A React app for tracking weekly activities and vibes.

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

### Preview
```bash
npm run preview
```

## Tech Stack
- React 18
- Vite
- Tailwind CSS
- PostCSS
